# Словко

Мобильное приложение для изучения английских, испанских, немецких, итальянских слов. Интерактивный словарь с тренажёром для эффективного запоминания.

## 📱 О приложении

Словко помогает расширить словарный запас через:
- 📖 Словарь с алфавитным указателем и фильтрацией по темам
- 🎯 Тренажёр для проверки знаний с системой прогресса
- 📊 Отслеживание сложности слов на основе ваших ответов
- 🔄 Адаптивный интерфейс для книжной и альбомной ориентации

## 🚀 Возможности

### Словарь
- Просмотр слов по алфавиту
- Фильтрация по темам
- Переключение между языками
- Быстрая навигация по алфавиту

### Тренажёр
- Выбор нескольких тем для тренировки
- Адаптивный подбор слов (сложные слова появляются чаще)
- Мгновенная проверка перевода
- Система прогресса: слова становятся легче при правильных ответах
- Автоматическое обновление сложности слов

### Выбор перевода
- Выбор нескольких тем для тренировки
- Изучение слов по выбору перевода из четырёх вариантов
- Мгновенная проверка перевода

### С Русского
- Выбор нескольких тем для тренировки
- Изучение слов переводов слов с русского на изучаемый
- Мгновенная проверка перевода

### Страница прогресса
- Обзор материала и свой статиски по темам
- Возможность просмотра тем и их слов
- Функция повтора тем прямо из страницы статиски, с выбором из трех вариантов (тренажёр, выбор перевода, с русского)
- Возможность удаления прогресса по темам, по словам, и по всем темам

## 🛠 Технологии

- Flutter - кросс-платформенная разработка
- SQLite - локальное хранение данных
- sqflite - работа с базой данных
